A web map application for biking in Olomouc developed in Web Cartography course.

KOlomouc - Bike Olomouc

You can access to the web map application HERE.


KOlomouc web map application was designed for the course Web Cartography. In Czech language, Kolo means bike and the area of interest was Olomouc. This application aimed to provide list of goal-orineted data layers, functions, and appealing design to share the idea regarding the potential of Olomouc to be as a biking city.
