GUI for Landslide Mapping in eCognition Software

The project focuses on developing an application for object-based image analysis (OBIA) in the field of landslide area mapping using eCognition software. The aim of the project is to create a ruleset and a graphical user interface (GUI) that allows users to apply our specific methodology for landslide mapping in our study area.
